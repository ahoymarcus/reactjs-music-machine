# Introdução aos React Hooks


Projeto de desenvolvimento front-end para estudar o novo React Hook. [^1]


## Dependências:

- Styled-components
- @testing-library/react
- MSW (Mock Service Worker) is a library for mocking requests to actual production URI
- React-js Hooks
- Proptypes
